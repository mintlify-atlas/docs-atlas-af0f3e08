---
title: "Software setup"
description: "Install Arduino IDE, add ESP32 board support, and install the required display libraries."
---

<Steps>
  <Step title="Install Arduino IDE">
    Download [Arduino IDE 2.x](https://www.arduino.cc/en/software) and install it.

    <Note>Arduino IDE 2.x is recommended. The legacy 1.x version may work but is not tested.</Note>
  </Step>

  <Step title="Add ESP32 board support">
    1. Open Arduino IDE → **File → Preferences**
    2. In "Additional boards manager URLs" paste:
       ```text
       https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json
       ```
    3. Go to **Tools → Board → Boards Manager**, search `esp32`, install **"esp32 by Espressif Systems"**
  </Step>

  <Step title="Install libraries">
    Go to **Tools → Library Manager** and install both:

    - `Adafruit GFX Library`
    - `Adafruit ST7735 and ST7789 Library`

    These correspond to the `#include` statements in the firmware:

    ```cpp
    #include <Adafruit_GFX.h>
    #include <Adafruit_ST7789.h>
    ```
  </Step>

  <Step title="Configure board settings">
    Go to **Tools** and set the following values:

    | Setting         | Value              |
    | --------------- | ------------------ |
    | Board           | ESP32C3 Dev Module |
    | USB CDC On Boot | **Enabled**        |
    | CPU Frequency   | 160 MHz            |
    | Upload Speed    | 921600             |

    <Warning>USB CDC On Boot must be set to **Enabled** — otherwise the serial connection will not work and flashing may fail.</Warning>
  </Step>
</Steps>
